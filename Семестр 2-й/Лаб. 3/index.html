Массивы

Массив (Array) - это упорядоченная коллекция элементов.

Элементы в массиве могут быть любого типа данных, включая числа, строки, объекты и т.д.

Чтобы добавить элемент в конец массива используется метод push()

Чтобы удалить элемент в конце массива используется метод pop()

Пример массива:

[1, 2, 3, 4, 5]
